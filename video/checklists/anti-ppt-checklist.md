# Anti-PPT Checklist

- [ ] 真实内容占比至少 70%—80%
- [ ] UI 与装饰素材合计不超过 10%
- [ ] 没有一页一个 Skill
- [ ] 没有一页一个观点的重复模板
- [ ] 没有白底知识卡或深色科技卡片轮播
- [ ] 没有纯文字页
- [ ] 空背景接近 0%
- [ ] 每镜有一个 55%—80% 的明确主视觉
- [ ] 背景持续含真实信息
- [ ] 终端和时间线没有长时间静止
- [ ] 每镜包含真实动作、切换、对比或结果
- [ ] 至少一组 Before / After
- [ ] 最终成片连续播放
- [ ] 结尾不是总结页
- [ ] 小黑人不是主角、不会口播或连续动画
- [ ] 抽帧联系表中相邻画面不是同构卡片
- [ ] 若任一项失败，已修改并重新渲染
