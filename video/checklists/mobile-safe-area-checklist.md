# Mobile Safe Area Checklist

- [ ] 画布 1080×1920，根容器 overflow hidden
- [ ] 核心文字 x 在 72—900 内
- [ ] 核心文字 y 在 140—1620 内
- [ ] 右侧 180px 没有关键文字或按钮
- [ ] 底部 300px 没有 CTA、选择题或关键数据
- [ ] 顶部 140px 没有主标题
- [ ] 手机/终端/数据窗口不超过 760×1250
- [ ] 分屏单窗不超过 360px，间距至少 24px
- [ ] 放大图片没有裁掉主体
- [ ] 大标题自动换行或缩小，没有溢出
- [ ] CTA 位于中下区域而非最底部
- [ ] 已生成危险区覆盖预览并逐镜检查
