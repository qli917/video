# Pre-production Checklist

- [ ] 只解决一个具体问题
- [ ] 目标观众和观看收益明确
- [ ] content brief 已填写
- [ ] source package 已建立
- [ ] 数据、旧视频、录屏和结果文件可访问
- [ ] 数据属于同一作品，口径和时间范围一致
- [ ] 公开参考只分析，不复制内容
- [ ] 至少一组真实失败与修改结果
- [ ] 隐私数据已脱敏
- [ ] 只提炼 1—3 个结论
- [ ] shot list 包含开始/结束状态和鼠标路径
- [ ] 口播实测可进入目标时长
- [ ] 没有用虚构 UI 填补素材缺口
