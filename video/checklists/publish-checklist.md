# Publish Checklist

- [ ] 最终 MP4 已真实渲染
- [ ] 时长符合 20—35 秒目标或有真实内容理由
- [ ] ffprobe 检查通过
- [ ] H.264、yuv420p、AAC、faststart
- [ ] 1080×1920、30fps、SAR 1:1、DAR 9:16
- [ ] 抽帧和联系表已生成
- [ ] anti-PPT 检查通过
- [ ] 手机安全区检查通过
- [ ] 无隐私、Token、账号或内部路径
- [ ] 标题 30 字内
- [ ] 话题不超过 5 个
- [ ] 封面文字手机端可读
- [ ] 评论回报真实且明确
- [ ] 关键词与口播一致
- [ ] 选择题或痛点问题具体
- [ ] 下一期承诺可执行
