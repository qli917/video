# Douyin Video Producer Skill

A Codex skill for producing short, evidence-driven Douyin videos with real content, strong retention, clear interaction design, and no PPT-style presentation.

## Core workflow

1. Build a real content package: problem, sources, conflict, process, result.
2. Use `agent-browser` for public reference research.
3. Use OpenAI Image for limited original hero visuals.
4. Use HyperFrames and Remotion for motion, editing, camera, and timing.
5. Use Whisper for voiceover timing and transcript checks.
6. Use video-use for frame review and anti-PPT quality control.
7. Use FFmpeg for mobile-compatible export.
8. Use create-promo-video for hook, evidence, payoff, and interaction structure.

## Installation

Clone this repository and point Codex to the root `SKILL.md`.

```bash
git clone git@github.com:qli917/video.git
```

## Non-negotiable rules

- Real content is the subject; skills are only tools.
- Default duration is 20–35 seconds.
- Every 2–3 seconds must add new information, action, contrast, or payoff.
- No Skill cards, whiteboard pages, UI carousels, or empty technology backgrounds.
- Every video must include a concrete comment prompt with a clear user reward.
- Every iteration is evaluated by retention and interaction metrics, not visual taste alone.
